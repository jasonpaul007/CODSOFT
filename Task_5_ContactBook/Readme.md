# Contact Management System

## Overview

The Contact Management System is a command-line application developed in Python that allows users to manage personal or professional contacts efficiently. Users can add, view, search, update, and delete contact information through a simple and interactive menu-driven interface.

Each contact contains the following details:

- Name
- Phone Number
- Email Address
- Address

## Features

- Add new contacts
- View all saved contacts
- Search contacts by name or phone number
- Update existing contact details
- Delete contacts
- Simple and user-friendly interface
- Menu-driven navigation

## Technologies Used

- Python 3

## Functionalities

### Add Contact
Allows users to save a new contact by entering:
- Name
- Phone Number
- Email Address
- Address

### View Contacts
Displays all saved contacts along with their names and phone numbers.

### Search Contact
Searches for a contact using:
- Name
- Phone Number

### Update Contact
Allows users to modify the details of an existing contact.

### Delete Contact
Removes a contact from the contact list using the phone number.
