# Rock Paper Scissors Game

## Overview

This project is a command-line implementation of the classic Rock Paper Scissors game using Python. The user plays against the computer, which makes random choices each round. The game determines the winner based on standard Rock Paper Scissors rules and keeps track of scores until the player chooses to exit.

## Features

- User-friendly menu-driven interface
- Play against the computer
- Random computer choice generation
- Winner determination based on game rules
- Score tracking across multiple rounds
- Input validation for incorrect entries
- Option to play multiple rounds

## Technologies Used

- Python 3
- Built-in `random` module

## Game Rules

- Rock beats Scissors
- Scissors beat Paper
- Paper beats Rock
- If both the player and computer choose the same option, the round is a tie
