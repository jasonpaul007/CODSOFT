# To-Do List Application

##  Overview

The To-Do List Application is a simple command-line program built using Python that helps users organize and manage their daily tasks. It provides basic task management functionality through an easy-to-use menu-driven interface.

##  Features

- Add new tasks
- View all tasks
- Update existing tasks
- Delete completed or unwanted tasks
- Simple and interactive command-line interface
- Continuous operation until the user chooses to exit

##  Technologies Used

- Python 3
